# Database_performance_comparison
Database performance comparison: local, cloud and docker mongoDB on Python.

## Install dependencies
Install dependencies:
`pip install pymongo`

## Consts
Create your own consts file with ips, ports and mongo cloud connection link

## Run program
Run main.py
